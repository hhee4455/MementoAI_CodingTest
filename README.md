# 1번 문제) x만큼 간격이 있는 n개의 숫자

## 문제 설명
함수 `solution`은 정수 `x`와 자연수 `n`을 입력 받아, `x`부터 시작해 `x`씩 증가하는 숫자를 `n`개 지니는 리스트를 리턴하는 함수입니다.

## 제한 조건
- `x`는 -10,000,000 이상, 10,000,000 이하인 정수입니다.
- `n`은 1 이상, 1000 이하인 자연수입니다.

## 입출력 예시
| x   | n   | 결과                          |
|-----|-----|-------------------------------|
| 2   | 5   | [2, 4, 6, 8, 10]              |
| 4   | 3   | [4, 8, 12]                    |
| -4  | 2   | [-4, -8]                      |
